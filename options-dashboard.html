🔌 API Pública para Desenvolvedores

Developer

Integre o corretor do SyntaxMentor no seu próprio site ou aplicação.

📘 Exemplo de uso: 
                    // Corrigir qualquer texto
                    const resultado = await SyntaxMentor.correct("texto com erro");
                    console.log(resultado.correctedText); // "texto sem erro"

                    // Corrigir elemento HTML
                    await SyntaxMentor.correctElement(document.getElementById("meu-texto"));

💡 Dica para desenvolvedores: A API está disponível globalmente como window.SyntaxMentor. Consulte a documentação completa.

Os dados são atualizados automaticamente conforme você usa a extensão.

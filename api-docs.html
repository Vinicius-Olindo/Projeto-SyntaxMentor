SyntaxMentor API

Documentação oficial para desenvolvedores

📦 Instalação

A API está disponível automaticamente quando a extensão SyntaxMentor está instalada.

// A API está disponível globalmente
window.SyntaxMentor.correct("texto");

🔧 Métodos Disponíveis

SyntaxMentor.correct(texto, options) async

Corrige um texto e retorna o resultado completo.

const resultado = await SyntaxMentor.correct("texto com erro");
console.log(resultado.correctedText);

SyntaxMentor.getSuggestions(texto, options) async

Retorna apenas a lista de sugestões.

const sugestoes = await SyntaxMentor.getSuggestions("testo");
// sugestoes = [{ original: "testo", suggestions: ["texto"] }]

SyntaxMentor.correctElement(elemento, options) async

Aplica correções diretamente em um elemento HTML editável.

await SyntaxMentor.correctElement(document.getElementById("meu-texto"));

SyntaxMentor.configure(options)

Configura a API (idioma, modo, etc).

SyntaxMentor.configure({
    language: "en-US",
    pickyMode: true
});

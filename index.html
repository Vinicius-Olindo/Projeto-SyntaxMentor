Instalacao local

Carregue a extensao em modo desenvolvedor.

O projeto usa Manifest V3 e pode ser testado diretamente no Chrome enquanto esta em desenvolvimento.

  1. Abra chrome://extensions.
  2. Ative o modo de desenvolvedor.
  3. Clique em Carregar sem compactacao.
  4. Selecione a pasta do projeto.
  5. Recarregue a extensao apos alterar scripts, CSS ou manifesto.

Seguranca

Permissoes objetivas e superficie reduzida.

O content script nao chama diretamente o LanguageTool; a comunicacao passa pelo background.

A chave de API e mantida em chrome.storage.session.

Scripts nao sao expostos em web_accessible_resources.

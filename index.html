Extensao Chrome para escrita assistida

Revise textos na web com menos friccao.

SyntaxMentor acompanha campos de texto, consulta o LanguageTool e entrega sugestoes claras para corrigir, ignorar ou aprender com cada revisao.

Como funciona

Um fluxo curto entre escrever e corrigir.

1

Digite normalmente

A extensao observa campos editaveis nas paginas permitidas e respeita o ritmo configurado.

2

Revise as sugestoes

O painel mostra erros, alternativas e explicacoes quando o modo aprendizado esta ativo.

3

Aplique ou ignore

Corrija uma sugestao, use correcao em lote ou adicione termos ao dicionario pessoal.

Fluxo principal

Feito para corrigir sem atrapalhar.

01

Revisao em tempo real

Detecta texto digitado, espera o ritmo configurado e apresenta sugestoes no painel lateral.

02

Dicionario pessoal

Permite ignorar nomes, termos tecnicos e expressoes que fazem sentido para o usuario.

03

Controle por site

Use blacklist, whitelist e atalhos para ajustar onde a extensao deve atuar.

04

Modos de uso

Inclui confirmacao, leitura, foco e aprendizado para contextos diferentes de escrita.

Instalacao local

Carregue a extensao em modo desenvolvedor.

O projeto usa Manifest V3 e pode ser testado diretamente no Chrome enquanto esta em desenvolvimento.

  1. Abra chrome://extensions.
  2. Ative o modo de desenvolvedor.
  3. Clique em Carregar sem compactacao.
  4. Selecione a pasta do projeto.
  5. Recarregue a extensao apos alterar scripts, CSS ou manifesto.

Permissoes

O necessario para revisar textos com transparencia.

storage

Guarda configuracoes, dicionario pessoal e listas de sites.

activeTab

Permite comunicar a extensao com a aba ativa quando necessario.

contextMenus

Adiciona atalhos no menu de contexto para revisar selecoes e ignorar palavras.

LanguageTool

Autoriza chamadas para api.languagetool.org, o motor externo de revisao.

Seguranca

Superficie reduzida e dados sob controle.

O content script nao chama diretamente o LanguageTool; a comunicacao passa pelo background.

A chave de API e mantida em chrome.storage.session.

Scripts nao sao expostos em web_accessible_resources.

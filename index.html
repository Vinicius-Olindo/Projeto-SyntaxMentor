Como funciona

Um fluxo curto entre escrever e corrigir.

1

Digite normalmente

A extensão observa campos editaveis nas páginas permitidas e respeita o ritmo configurado.

2

Revise as sugestões

O painel mostra erros, alternativas e explicações quando o modo aprendizado está ativo.

3

Aplique ou ignore

Corrija uma sugestao, use correção em lote ou adicione termos ao dicionário pessoal.

Fluxo principal

Instalação local

Carregue a extensão em modo desenvolvedor.

O projeto usa Manifest V3 e pode ser testado diretamente no Chrome enquanto está em desenvolvimento.

  1. Abra chrome://extensions.
  2. Ative o modo de desenvolvedor.
  3. Clique em Carregar sem compactação.
  4. Selecione a pasta do projeto.
  5. Recarregue a extensão após alterar scripts, CSS ou manifesto.

Permissões

O necessário para revisar textos com transparência.

storage

Guarda configurações, dicionário pessoal e listas de sites.

activeTab

Permite comunicar a extensão com a aba ativa quando necessário.

contextMenus

Adiciona atalhos no menu de contexto para revisar seleções e ignorar palavras.

LanguageTool

Autoriza chamadas para api.languagetool.org, o motor externo de revisão.

Segurança

Superfície reduzida e dados sob controle.

O content script não chama diretamente o LanguageTool; a comunicação passa pelo background.

A chave de API é mantida em chrome.storage.session.

Scripts não são expostos em web_accessible_resources.
